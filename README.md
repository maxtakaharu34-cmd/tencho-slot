# てんちょースロット

ミリオンゴッド神々の凱旋を参考にした、てんちょーさん PR 用の本格パチスロ風 Web アプリ。

- 単一 HTML / Vanilla JS / Canvas / WebAudio
- ビルド・依存ゼロ
- PC キーボード + スマホタッチ対応

## 遊ぶ

GitHub Pages 公開版:
<https://maxtakaharu34-cmd.github.io/tencho-slot/>

ローカルで遊ぶ場合は `index.html` をブラウザで開くだけ。

## 操作

| キー / ボタン | 動作 |
| --- | --- |
| Space / レバー | スピン開始 |
| Z / 左STOP | 左リール停止 |
| X / 中STOP | 中リール停止 |
| C / 右STOP | 右リール停止 |

## モード

- **通常**: 7 揃い (1/20) で CZ、特殊図柄 (1/100) で上位 AT 直行
- **CZ「暗黒星雲チャンス」**: 押し順ベル 10 回チャレンジ、ベル 1/10 / スイカ・チェリーで AT 確定
- **AT「暗黒星雲モード」**: 50G、純増 +3 枚、ベル 15 枚、押し順ベル 1/5
- **復活 CZ「暗黒星雲復活チャンス」**: 5G 以内に子役で AT 継続
- **上位 AT「超闇暗黒星雲モード」**: 100G、チェリー / スイカで +10G、転落 1/199
